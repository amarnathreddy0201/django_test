Specification of this Project

Specification:

project with the name “TradingProject”.

Django app within the project “MainApp”.

MainApp should have a view to accepting a CSV file and timeframe(will be a number).

Store the CSV file on the Django server. You have to read a CSV file using the fastest option available. Store the data present in the file in the python list of candle objects.

The candle object will have attributes: [id, open, high, low, close, date]

You have to convert the list of candles that will be one minute into a given timeframe. [Use async operations] Store this converted data into a JSON file and store it in the file system.

In the response, the user should get the option to download the JSON file.
